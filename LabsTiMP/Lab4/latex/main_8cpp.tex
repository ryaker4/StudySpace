Программа предоставляет пользовательский интерфейс для работы с шифром табличной маршрутной перестановки.

Программа позволяет\+:
\begin{DoxyEnumerate}
\item Зашифровать текст с использованием заданного ключа (количество столбцов)
\item Расшифровать ранее зашифрованный текст
\item Выйти из программы 
\end{DoxyEnumerate}
