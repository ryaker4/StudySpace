\doxysubsection{Подробное описание}
Класс для шифрования методом табличной маршрутной перестановки 

Реализует шифр табличной маршрутной перестановки для русского текста. Ключом является количество столбцов таблицы. Маршрут записи\+: по горизонтали слева направо, сверху вниз. Маршрут считывания\+: сверху вниз, справа налево. \begin{DoxyWarning}{Предупреждения}
Реализация поддерживает только русские буквы и пробелы 
\end{DoxyWarning}
